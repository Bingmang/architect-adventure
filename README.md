## 前言

本仓库用于保存我在学习架构知识路上发现的学习资源，供大家参考学习。

## 感悟

还处于学生时期的程序员们可以多花点时间在计算机基础上（操作系统、计算机网络、算法与数据结构、设计模式等），不要急着学习框架、开源项目，核心思想的养成是你以后的核心竞争力。

我们要做到除[“机心”](https://baike.baidu.com/item/%E6%9C%BA%E5%BF%83)，是我们选择工具，而不是工具选择我们，多思考背后的原理。
